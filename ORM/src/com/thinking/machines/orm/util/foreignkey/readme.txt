ForeignKeyInformation.java -

It is person which is responsible to represent ForeignKey Information,

It provides three information i.e

Colum name of table
Foriegn Table Name 
Foreign Table Column

eg:
there is student table which has
course_code field mapped to Course Table code column So

Colum name of table  -> course_code
Foreign Table Name   -> Couse
Foreign Table Column -> code

All getter & setters for all information
